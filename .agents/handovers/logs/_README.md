# handovers/logs/ — Handover History Archive

> This directory stores completed handover documents organized by date.
> Contents from `to_<role>.md` are appended here upon completion.

---

## File Naming Convention

```
YYYY-MM-DD_to_<role>.md
```

Examples:
- `2026-03-27_to_backend_dev.md`
- `2026-03-28_to_frontend_dev.md`

---

## Recording Method

- If the file for that date **already exists**: **Append** below existing content
- If the file for that date **does not exist**: Create a new file

---

> **Files in this directory are a read-only archive.**
> Used for tracking handover history and reviewing "who requested what, and when."
