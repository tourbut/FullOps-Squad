[Design System Reference for LLMs]
이 파일은 AI 에이전트(LLM)가 디자인 시스템을 이해할 수 있도록 프롬프트에 주입하기 위한 평문(Text/Markdown) 레퍼런스입니다.
이 파일은 `DESIGN.md §4 표준 디자인 소스 세트`의 "토큰 단일 진실 공급원(SSOT)"이다. 프론트엔드 구현 시 모든 색상·타이포·간격은 아래 토큰 값만 사용한다(하드코딩 금지).

## 색상 토큰 (Color Tokens)

### Light Mode
| Token | Value | Usage |
|-------|-------|-------|
| --color-primary | #3B82F6 (Blue-500) | 주요 CTA, 활성 네비게이션 |
| --color-primary-hover | #2563EB (Blue-600) | Primary 호버 상태 |
| --color-secondary | #6B7280 (Gray-500) | 보조 텍스트, 비활성 요소 |
| --color-bg | #FFFFFF | 페이지 배경 |
| --color-surface | #F9FAFB (Gray-50) | 카드, 패널 배경 |
| --color-border | #E5E7EB (Gray-200) | 경계선, 구분선 |
| --color-error | #EF4444 (Red-500) | 에러 상태 |
| --color-success | #22C55E (Green-500) | 성공 상태 |
| --color-warning | #F59E0B (Amber-500) | 경고 상태 |

### Dark Mode
| Token | Value | Usage |
|-------|-------|-------|
| --color-primary | #60A5FA (Blue-400) | 주요 CTA |
| --color-bg | #111827 (Gray-900) | 페이지 배경 |
| --color-surface | #1F2937 (Gray-800) | 카드, 패널 배경 |
| --color-border | #374151 (Gray-700) | 경계선 |
| --color-text | #F9FAFB (Gray-50) | 주요 텍스트 |

## 타이포그래피 (Typography)

| Level | Size | Weight | Line Height |
|-------|------|--------|------------|
| H1 | 2rem (32px) | Bold (700) | 1.25 |
| H2 | 1.5rem (24px) | Semibold (600) | 1.3 |
| H3 | 1.25rem (20px) | Semibold (600) | 1.4 |
| Body | 1rem (16px) | Regular (400) | 1.5 |
| Small | 0.875rem (14px) | Regular (400) | 1.5 |
| Caption | 0.75rem (12px) | Regular (400) | 1.5 |

## 간격 시스템 (Spacing)

- 기본 단위: 4px
- 주요 간격: 4, 8, 12, 16, 24, 32, 48, 64px
- 컴포넌트 내부 패딩: 12~16px
- 카드 간 마진: 16~24px
- 섹션 간 마진: 32~48px
